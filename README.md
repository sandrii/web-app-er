# web-app-er
=======
## fastapi-html


## Quickstart

```
# Clone this repo
git clone https://github.com/sandrii/web-app-er.git

# Install the required dependencies
cd web-app-er
poetry install

# Start the HTML app
poetry run uvicorn src.html:app --reload
```

Navigate to [127.0.0.1:8000/form](http://127.0.0.1:8000/form).
